# 股票篩選網站

一個可以讓使用者根據股價範圍和股票類別篩選股票的網站。

## 功能特色

- 📊 **股價篩選** - 設定最低價和最高價來過濾股票
- 🏷️ **類別篩選** - 選擇科技、金融、製造、電信等類別
- 🔍 **即時篩選** - 按下篩選按鈕立即顯示符合條件的股票
- 🎨 **現代化介面** - 簡潔美觀的使用者介面
- 📱 **響應式設計** - 適配各種螢幕尺寸

## 技術架構

- **前端框架**: React Native + Expo
- **導航**: Expo Router
- **資料庫**: Supabase (PostgreSQL)
- **語言**: TypeScript

## 設置說明

1. **安裝依賴**
```bash
npm install
```

2. **配置 Supabase**

   更新 `.env` 檔案中的 Supabase 連線資訊：
   ```
   EXPO_PUBLIC_SUPABASE_URL=你的_supabase_網址
   EXPO_PUBLIC_SUPABASE_ANON_KEY=你的_supabase_匿名金鑰
   ```

3. **啟動開發伺服器**
```bash
npm run dev
```

4. **開啟網頁**

   在瀏覽器中打開顯示的網址（通常是 http://localhost:8081）

## 資料庫結構

### stocks 資料表

| 欄位 | 類型 | 說明 |
|------|------|------|
| id | uuid | 主鍵 |
| code | text | 股票代碼 |
| name | text | 股票名稱 |
| price | numeric | 股價 |
| category | text | 股票類別 |
| created_at | timestamptz | 建立時間 |

### 範例資料

已預先載入 15 支台灣股票資料，包含：
- 科技股：台積電、鴻海、聯發科等
- 金融股：國泰金、中信金、兆豐金等
- 製造股：中鋼、台塑、南亞等

## 使用方式

1. **設定股價範圍**
   - 輸入最低價（選填）
   - 輸入最高價（選填）

2. **選擇股票類別**
   - 點擊類別標籤進行選擇
   - 可以選擇多個類別
   - 已選擇的類別會以綠色顯示

3. **套用篩選**
   - 點擊「套用篩選」按鈕
   - 下方會顯示符合條件的股票列表

4. **重置篩選**
   - 點擊「重置」按鈕清除所有篩選條件
   - 顯示所有股票

## 股票卡片資訊

每張股票卡片顯示：
- 股票代碼（左上）
- 股票名稱（左下）
- 當前股價（右上，綠色）
- 股票類別（右下，灰色標籤）

## 新增更多股票

可以透過 Supabase 控制台或 SQL 指令新增更多股票：

```sql
INSERT INTO stocks (code, name, price, category)
VALUES ('0050', '元大台灣50', 145.50, '基金');
```
